# Requirements: Omaxe Eternity 2 Website Optimization

**Defined:** 2026-04-11
**Core Value:** To seamlessly convert high-intent real estate investors into active leads through frictionless communication channels (WhatsApp and Calls) while maintaining perfect search engine visibility.

## v1 Requirements

### Technical SEO (SEO)

- [ ] **SEO-01**: Fix Google Search Console "Page with redirect" and "Redirect error" mapping.
- [ ] **SEO-02**: Ensure canonical tags correctly match the actual deployment URLs to resolve indexation confusion.
- [ ] **SEO-03**: Remove `.html` extensions from internal linking (navigation menus, footer, and sitemap.xml) to match server cleanly.

### Conversion Rate Optimization (CRO)

- [ ] **CRO-01**: Optimize WhatsApp buttons (make them stickier, more prominent, and clearly visible on mobile viewports).
- [ ] **CRO-02**: Improve Direct Call buttons to ensure immediate response from high-intent buyers.
- [ ] **CRO-03**: Standardize high-intent Call-to-Action (CTA) text (e.g., "Get Price List on WhatsApp", "Speak to an Expert").

### Lead Quality (LEAD)

- [ ] **LEAD-01**: Refine website copy surrounding the lead magnets to better target serious plot investors rather than casual browsers.

## v2 Requirements

### Analytics & Tracking

- **TRK-01**: Implement Google Tag Manager event tracking specifically for WhatsApp button clicks vs Form Submissions to analyze lead paths.

## Out of Scope

| Feature | Reason |
|---------|--------|
| CMS Migration | Site performs well on static deployment; migrating to a CMS adds unnecessary overhead right now. |
| Database Engine | Contact forms seamlessly pass data to Google Sheets; building a relational DB is over-engineered for our needs. |
